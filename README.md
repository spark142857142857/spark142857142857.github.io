# GitHub Pages portfolio

## Personalize

Open `index.html` and replace every placeholder:

- `Your Name`, `YN`, location, introduction, and email
- GitHub and LinkedIn links
- Three project cards and activity entries
- Each `href="#"` for a project’s repository or live demo

## Publish for free

1. Create a **public** GitHub repository named exactly `YOUR-GITHUB-USERNAME.github.io`.
2. Upload `index.html` and `styles.css` from this folder to the repository root and commit.
3. In the repository, open **Settings → Pages**. Under **Build and deployment**, choose **Deploy from a branch**, select `main` and `/ (root)`, then save.
4. After GitHub finishes deploying, visit `https://YOUR-GITHUB-USERNAME.github.io`.

You can update the site later by editing the files and pushing a new commit.
